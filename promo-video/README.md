# SportBook Ultimate Promo

15-second vertical Remotion promo for SportBook Ultimate.

## Commands

```console
npm i
npm run dev
npm run lint
npm run render
npm run still
```

## Composition

- `SportBookUltimatePromo`
- `1080x1920`
- `30 fps`
- `450 frames`

## Source map

- `src/constants.ts` brand, copy, timing map, audio cues, safe areas
- `src/components/*` reusable phone, dashboard, overlays, animated text
- `src/scenes/*` pain, solution, player flow, operator power, CTA
- `src/Composition.tsx` scene assembly
